# Luxo\_Spk\_Enclosure

### Description :

* Speaker Enclosure of 49.7JQ01A002, which is extracted part from Samsung Freestyle Projector 

### Known Issues :

* Slightly tight to assembly (may vary printing condition)
* To add grill mesh on top cover, should to edit "top\_cover" part to add more space.
* Hole of USB-C receptacle is too large.
* Rubber feet is protrude to the sides.

### Parts Lists :

* 3D printed parts
* 2x Screw - M3 x 6mm
* 2x Insert Nut - M3 x 3mm
* 4x Screw - M3 x 8mm
* 4x Nut - M3 
* Speaker Unit - 49.7JQ01A002
* Bluetooth Audio Module - MH-M38
* Locker Switch - KCD11-101mini
* 4x Rubber feet
